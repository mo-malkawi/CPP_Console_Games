# C++ Console Games
Two console-based games built with C++.

## 1. Rock Paper Scissors
A console-based Rock Paper Scissors game built with C++.

## How it works
- Choose how many rounds to play
- Pick Rock, Paper, or Scissors each round
- The computer picks randomly
- Console turns green if you win the round, yellow for draw, red if computer wins (with a beep)
- Final results screen appears after all rounds with overall winner
- Play again option after the game ends
- Supports 1 to 10 rounds

## Technologies
C++, Enums, Structs, rand(), srand()

## 2. Math Quiz
A console-based math quiz game built with C++

## How it works
- Choose how many questions to solve
- Choose questions difficulty Easy, Med, Hard, Mix
- Choose operation type Add, Sub, Mul, Div, Mix
- Console turns green if answer is correct, red if answer is wrong
- Final result screen appears after solving all questions
- Play again option after quiz ends

## Technologies
C++, Enums, Structs, rand(), srand()
