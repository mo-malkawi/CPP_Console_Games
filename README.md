# Rock Paper Scissors

A console-based Rock Paper Scissors game built with C++.

## How it works
- Choose how many rounds to play
- Pick Rock, Paper, or Scissors each round
- The computer picks randomly
- Console turns green if you win the round, yellow for draw, red if computer wins (with a beep)
- Final results screen appears after all rounds with overall winner
- Play again option after the game ends
- Supports 1 to 10 rounds

## Technologies
C++, Enums, Structs, rand(), srand()
